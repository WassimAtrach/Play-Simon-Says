# 🕹️ Simon Game

A web-based version of the classic electronic memory game. The game creates a series of tones and lights and requires the user to repeat the sequence. If the user succeeds, the series becomes progressively longer and more complex.



## 🚀 Live Demo
You can play the game live here:
**[Play Simon Says](https://wassimatrach.github.io/Play-Simon-Says/)**

---

## 🛠️ Built With
* **HTML5** - For the structure of the game.
* **CSS3** - For the styling and animations (hover effects, shadows).
* **JavaScript** - For the game logic.
* **jQuery** - Used for DOM manipulation and event handling.

## 🎮 How to Play
1. **Start:** Press any key on your keyboard to start the game.
2. **Watch:** The game will flash a button and play a sound.
3. **Repeat:** Click the buttons in the same sequence shown.
4. **Level Up:** Each time you get the sequence right, the game adds one more step.
5. **Game Over:** If you click the wrong button, the game ends. Press any key to restart!

---

## 🧠 Key Features
* **Dynamic Sequences:** Randomly generated patterns that increase in difficulty.
* **Audio Feedback:** Unique sounds for each button and a "wrong" sound for mistakes.
* **Visual Effects:** Smooth fade-in/fade-out animations and button press styling.
* **Responsive Design:** Playable on various screen sizes.

## 📂 Project Structure
* `index.html` - The main entry point.
* `game.js` - The logic controlling the patterns, user input, and sound.
* `styles.css` - Custom styling for the 2x2 grid.
* `/sounds` - Audio files for the game experience.
